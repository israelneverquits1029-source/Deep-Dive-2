# Deep Dive

Deep Dive is a simple 2D platformer where I play as a knight exploring underwater caves, collecting coins while avoiding obstacles.

This game was created as part of a hackathon submission.

## Gameplay

- Move left/right
- Jump between platforms
- Collect all coins to win
- Avoid obstacles

When all coins are collected, a win screen appears with a restart option.

## Built With

- Godot Engine
- GDScript
- Pixel Art Assets

## How To Play

Play the game here:
